## 專案說明
* 網站主要使用 Vue.js 及 Vue Cli 架設而成
* 加入 ESLint Airbnb 撰寫風格
* 透過 mitt 傳遞跨元件資料

## 頁面功能
* 基金條件搜尋： 關鍵字highlight搜尋、多重搜尋功能
* 比較表： 1-3檔基金圖表呈現與比較
* 觀察名單： 基金加入觀察清單
* 申購確認： 驗證與確認

## 使用技術
- Vue3
- Vue Cli
- Vue Component
- Vue Router
- chart.js
- Swiper
- Vue loading-overlay
- mitt
- VueSweetalert
